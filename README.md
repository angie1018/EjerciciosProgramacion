# Ejercicios de Programación

Este repositorio reúne varios ejercicios desarrollados en Java como parte del taller de nivelación.  
Cada actividad está organizada por carpetas y contiene su respectivo archivo `.java` para facilitar su revisión y ejecución.

---

## Contenido

### • Calculadora Básica
Implementación de una calculadora por consola que permite realizar operaciones elementales como suma, resta, multiplicación y división, incluyendo validación de entradas y manejo de errores (por ejemplo, división entre cero).

### • Contador de Vocales y Consonantes
Programa que analiza una cadena de texto e identifica cuántas vocales y consonantes contiene, ignorando espacios y caracteres no alfabéticos.

### • Invertir una Cadena
Ejercicio simple que recibe una cadena y devuelve su versión invertida, reforzando manipulación de strings.
